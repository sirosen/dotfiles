= dotfiles

A dotfiles repo using GNU stow

Always changing, sometimes improving

== Usage

Place this repo in your home directory, with no intermediate dir structure.
cd into this directory (stow automatically targets the parent dir by default).

== Unpack Configs

  stow vim
  stow zsh
  stow tmux
  stow git
  stow bin
  stow cheatsheets
  stow helix ~/.config/
